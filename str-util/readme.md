
# Str Util

## Cents

Converts between cents (int) and dollar strings "0.00" or "$0.00" without loss
of precision.

| Name | Description |
| - | - |
| dollarString(cents) | "$0.00" |
| string(cents) | "0.00" |
| dollarStringToCents(str) |  "1.00" or "$1.00" => 100 |
